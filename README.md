Len UI 是一个小型的 UI 框架，提供了一些常用组件，适合桌面端使用。

组件：

- Button 按钮
- Checkbox 多选框
- Collapse 折叠面板
- Dialog 对话框
- Form 表单
- Input 输入框
- Radio 单选框
- Switch 开关
- Tab 标签页
- Tooltip 文字提示

## 介绍

这是为 Vue 打造的一个 轻量级 UI 框架，参考了 ELement 进行实现。

## 开始使用

1. 安装 Len UI

```
npm install len-ui
```

或者

```
yarn add len-ui
```

2. 引入 Len UI

```Javascript
import LenUI from 'len-ui'

import 'len-ui/dist/len-ui.css'
```
