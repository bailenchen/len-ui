# Len UI

![](https://img.shields.io/badge/language-JavaScript-yellow.svg)
![](https://img.shields.io/badge/license-MIT-000000.svg)

Len UI 是一个小型的 UI 框架，提供了一些常用组件，适合桌面端使用。

组件：

- Button 按钮
- Radio 单选框
- Checkbox 多选框
- Input 输入框
- Switch 开关
- Form 表单
- Tab 标签页
- Tooltip 文字提示
- Dialog 对话框
- Collapse 折叠面板

## 关于 Len

这是为 Vue 打造的一个 轻量级 UI 框架，参考了 ELement 进行实现。
